# Branch

## 一、事件级信息

| Branch名称           | 中文解释                                                       |
| ------------------ | ---------------------------------------------------------- |
| EVENTNUMBER        | 事件编号，用于唯一标识对撞实验中记录的每一个独立事件                                 |
| EVENTTYPE          | 事件类型，标识事件的物理过程类别，用于区分不同的衰变通道或物理过程                          |
| RUNNUMBER          | 运行编号，标识数据采集期间的特定运行周期                                       |
| FillNumber         | 填充编号，标识对撞机的一次束流填充周期，即从注入质子束流到束流耗尽的时间段                      |
| GPSTIME            | 全球定位系统时间，记录事件发生时的精确时间戳，用于时间同步和事件排序                         |
| BUNCHCROSSING_ID   | 束团交叉编号，标识对撞机中两个束团发生对撞的具体位置编号                               |
| BUNCHCROSSING_TYPE | 束团交叉类型，标识束团交叉的性质，如是否为真实对撞或测试交叉 |
| Decision           | 决策标志，表示该事件是否通过了特定的选择条件或筛选算法                                |
| nPVs               | 主顶点数量，记录事件中重建出的主顶点总数，反映事件的复杂程度                             |
| PV_SIZE            | 主顶点规模，记录主顶点相关数据结构的大小                                       |
| ODINTCK            | 在线数据信息系统触发配置键，标识在线数据处理的触发器配置版本                             |

## 七、粒子运动学变量

| Branch名称    | 中文解释                                 |
| ----------- | ------------------------------------ |
| M           | 不变质量，根据粒子四动量计算得到的相对论不变质量，用于粒子鉴别和物理分析 |
| P           | 总动量，粒子在三维空间中的动量大小                    |
| log_P       | 总动量对数，总动量的自然对数值，用于扩展动态范围和改善分布特性      |
| E           | 能量，粒子的总能量，包括静止能量和动能                  |
| PT          | 横向动量，粒子动量在垂直于束流方向平面上的投影分量            |
| log_PT      | 横向动量对数，横向动量的自然对数值                    |
| PX          | 横向动量X分量，粒子动量在探测器坐标系X方向的分量            |
| PY          | 横向动量Y分量，粒子动量在探测器坐标系Y方向的分量            |
| PZ          | 纵向动量分量，粒子动量沿束流方向的分量                  |
| ETA         | 赝快度，描述粒子运动方向的角坐标，定义为负自然对数正切半极角       |
| PHI         | 方位角，粒子动量在垂直于束流平面上的投影与X轴的夹角           |

## 二十三、触发器匹配变量

| Branch名称 | 中文解释                           |
| -------- | ------------------------------ |
| _TIS     | 触发独立于信号，表示触发决策与特定信号候选无关的触发匹配类型 |
| _TOS     | 触发依赖于信号，表示触发决策由特定信号候选触发的触发匹配类型 |

**说明**：带有_TIS后缀的变量表示该粒子或候选与触发器决策独立匹配，即触发器决策不依赖于该特定信号候选。带有_TOS后缀的变量表示该粒子或候选与触发器决策信号匹配，即触发器决策是由该特定信号候选触发的。这两种触发匹配类型在物理分析中用于正确计算触发效率和处理触发偏差。
