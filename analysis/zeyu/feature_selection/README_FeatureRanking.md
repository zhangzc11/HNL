# HNL 物理特征筛选工具 (Feature Ranking Tool) 
## 项目背景 
本项目用于在 LHCb HNL 寻找分析中，从上千个原始变量中筛选出对模拟信号 (Signal MC) 和本底 (Data) 具有高区分度的物理变量。 
## 功能描述
脚本 `find_important_features1.py` 实现了以下功能：
1. 自动过滤变量：剔除所有非物理变量（如 `_Key`, `_ID`）以及 Data 中不存在的真值变量，选取Data和Signal MC中共有变量
2. 遍历 `ntuple_20260228` 目录下所有不同质量（5-50GeV）和寿命的 MC 样本。
3. 利用 **KS 检验 (Kolmogorov-Smirnov test)** 计算每个变量下Data和Signal MC的区分度。KS值的范围在 0 到 1 之间，KS值越大表明区分度越大，0.4以上模型区分效果就算比较理想。
4. 平均排名：由高到低输出所有不同质量（5-50GeV）和寿命样本的平均区分度 (`mean`) 和稳定性 (即标准差`std`)，结果保存至 `global_feature_ranking.csv`。 
## 关键结论 
通过分析发现：核心物理变量（如不变质量 `_M` 和横动量 `_PT`）在所有质量点均表现出极高的区分度（>0.5）。 
## 说明
1. 在分析时，我们将优先从统计排名前列的变量中，选取具有明确物理定义的量进行切割优化，而非盲目使用所有高得分变量，以增强分析的可解释性。
2. LHCb 中的 HNL 寻找通常分为 SS  和 OS 两种电荷组合拓扑，通过对 SS 和 OS 样本分别运行筛选工具，我们可以为不同的拓扑分类制定针对性的特征子集。
3. 可以针对某个特定变量进行进一步的分析：在不同质量和寿命条件下KS值和分布是如何变化的，探究模型在全参数空间（质量 $m_N$ 与寿命 $\tau_N$）的性能稳定性（例如某些变量只在低质量区有区分度），这有助于我们判断是否需要分段选择，以便后续的模型训练。
